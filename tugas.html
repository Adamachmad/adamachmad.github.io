Cara Download dan Install Python Pada Windows

Bahasa pemrograman Python diciptakan oleh Guido Van Rossum dan dikenalkan sejak tahun 1991. Secara umum Python berfungsi untuk melakukan pengembangan website, software, system scripting, dan matematika. Bahasa pemrograman Python termasuk yang mudah dipahami dan dipelajari bahkan oleh pemula. Struktur bahasanya yang tidak terlalu rumit dibandingkan bahasa pemrograman lainnya menjadikannya bahasa pemrograman yang populer. Penulisan kode yang digunakan untuk memberikan suatu perintah dalam program juga tidak terlalu panjang, jadi developer akan lebih fokus pada pembuatan programnya.

Selain digunakan untuk mengembangkan program, Python banyak digunakan untuk keperluan data science. Mengapa? Python memiliki banyak package dan library yang memudahkan aktivitas tersebut. Penggunaannya pun cukup mudah hanya dengan memanggil library terlebih dahulu sebelum menuliskan kode selanjutnya. Sebagai pemula, kau bisa mencoba belajar Python untuk melakukan analisis sederhana menggunakan mini dataset. Sebelum mempelajarinya, tentu kamu perlu mendownload dan install Python terlebih dulu. Bagaimana caranya? Nah, di artikel kali ini kita kan membahas cara download dan install Python di berbagai sistem operasi. Yuk, simak di bawah ini!

Download Python

kamu tidak mempunyai file instalasi Python sebelumnya, kamu perlu mendownloadnya. File instalasi Python dapat di download di website resminya di python.org. Di website tersebut tersedia dua versi Python yaitu Python 2 dan Python 3. Apa bedanya? Python versi 2 merupakan versi yang terkenal dan banyak digunakan, sedangkan Python versi 3 merupakan pengembangan lanjutan dari Python versi 2. Python 3 lebih canggih karena memiliki banyak fitur baru dan perbaikan bug. Sistem security pada Python 3 juga lebih baik. Perbedaan penulisan kode yaitu misalnya fungsi print pada Python 2 dituliskan printHello World! sedangkan pada Python 3 ditambahkan tanda kurung menjadi print(œHello World!). Cara download Python yaitu sebagai berikut:
1. Buka python.org di browser
2. Pilih tombol Download
3. Pilih sesuai sistem operasi yang kamu gunakan seperti Windows, Linux atau Mac
4. Klik link pada versi Python yang diinginkan
5. Secara otomatis file akan terdownload

Cara Install di Windows

Windows merupakan sistem operasi yang dikeluarkan oleh Microsoft. Dengan tampilan yang user-friendly membuat Windows menjadi yang paling populer bagi pengguna komputer atau laptop. Bagi pengguna Windows berikut cara install Python yang sudah di download sebelumnya:
1. Buka file install yang sudah di download dari website resmi Python dengan double click
2. Pilih "Install for all users" agar bisa digunakan oleh semua user di komputer atau laptop
3. Konfigurasi Python agar dikenali oleh sistem operasi dengan checklist Add Python 3.7 to PATH atau versi yang kamu install
4. Klik Install Now
5. Jika instalasi berhasil akan muncul tulisan Setup was Successful
Selanjutnya kita bisa uji coba Python dengan menjalankan command prompt (CMD) dan tuliskan python untuk masuk ke mode Python. Kita juga bisa menjalankan Python menggunakan Python Shell dengan mencari di Search pada Windows. Klik kanan kemudian run as administrator pada Python Shell.
